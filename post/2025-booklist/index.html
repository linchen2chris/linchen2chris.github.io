2025年阅读的书书单

ING 1 《Efficient Go》

DONE 2 《Pathless Path》

DONE 3 《认识圣经》

DONE 4 《哥德尔，埃舍尔和巴赫—集异璧之大成》

终于读完了这个巨著，打开了我的新世界，关于逻辑，悖论，智能的很多深入的讨论, 在当前AI火热的时节, 书中30多年前的很多内容仍然没有过时. 哥德尔不完全性等信息在启发我们未来AI的极限可能在哪里

DONE 5 《漫画理想国》

DONE 6 《极简物理》

DONE 7 《知识的边界》

DONE 8 《解经式讲道》

DONE 9 《禅与摩托车的维修艺术》

一本很好的哲学入门书, 涉及认识论中古典主义,浪漫主义. 逻辑和美, 其中关于良质的讨论更是让人心往神怡.
